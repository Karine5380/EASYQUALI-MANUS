# Veille Formation Professionnelle — EasyQuali

Ce répertoire contient les veilles quotidiennes de la formation professionnelle, conformes aux exigences Qualiopi (Indicateurs 23, 24, 25, 26).

## Structure

Chaque veille est organisée dans un dossier daté au format `AAAA-MM-JJ` contenant :

| Fichier | Description |
|---|---|
| `veille_flash.md` | Synthèse Markdown complète des actualités du jour |
| `script_podcast.md` | Script complet du podcast de 15 minutes |
| `description_youtube.md` | Description YouTube optimisée SEO avec timestamps |

## Périmètre

Formation professionnelle pour adultes uniquement :
- CFA (Centres de Formation d'Apprentis)
- Organismes de formation professionnelle
- Bilan de compétences
- VAE (Validation des Acquis de l'Expérience)

## Indicateurs Qualiopi couverts

- **Indicateur 23** : Veille Légale et Réglementaire
- **Indicateur 24** : Veille des Emplois et Métiers
- **Indicateur 25** : Veille Pédagogique et Technologique
- **Indicateur 26** : Accessibilité et Handicap

## Règles de publication

- Sources françaises officielles uniquement
- Actualités datant du jour de publication uniquement
- Lien de publication unique par jour (format `/veille/AAAA-MM-JJ`)
